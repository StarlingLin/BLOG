# 设置随机数范围

比方说要生成 0-99 间的随机数，我们可以:

rand() % 100

如果是 1-100 间:

rand() % 100 + 1

如果是 1-666 间:

rand() % 666 + 1

由此可以得到，在 a-b 间:

a + rand() % (b-a+1)
