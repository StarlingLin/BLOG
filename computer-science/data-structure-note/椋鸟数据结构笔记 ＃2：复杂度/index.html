# 复杂度

复杂度是衡量算法好坏的一种方式。一般分为时间复杂度和空间复杂度，分别用于衡量一个算法在运行时间长短和占据内存空间多少两方面的优劣。

一般我们考察复杂度就是找到相关的代数式，将常数全部忽略，将非最高次项全部忽略，在用括号括起来，前面写一个大 O。这就是复杂度的大 O 表示法。

比方说，对于代数式 3N^2+2N+4，我们只取 N^2；对于 2N+4M+1，我们只取 M+N。
